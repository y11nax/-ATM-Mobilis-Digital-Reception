# ATM Mobilis — Digital Reception

> Flask-based visitor management system replacing manual paper logbooks at ATM Mobilis's reception desk.

## Features

- Register visitors with identity, department, host, and access level
- Live dashboard with occupancy tracking and recent entries
- Searchable records with entry/exit timestamps
- Printable visitor badge with barcode
- Analytics by department and daily volume
- CSV export

## Stack

- **Python / Flask** — routing and application logic
- **SQLite** — local persistent storage
- **HTML / CSS** — reception-optimised interface

## Run

```bash
pip install flask
python app.py
```

App runs at `http://127.0.0.1:5000`
